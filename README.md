# Introduction
This is my personal dotfiles managed by stow. It's pursuing of extremely minimal(in my personal..)

It can be only worked on Mac OS.

# Features

- IDE env using neovim with useful plugins
- Add new terminal in one window using Tmux and perfect interaction with neovim
- Only essential plugins on zsh
- nvim lsp settings
- nvim debug settings
- paper theme for protect eyesights
- font can use whatever you want, but in my recommendation is using the iosevka light.

### Requirements

- mac OS (homebrew)
- wezterm
- SDKMan (JDK version manager)
- uv (Python manager)
- bob (neovim version manager)
- zplug
- zsh
- neovim: +0.12 nightmare
- zoxide
- fzf
