# bat

[bat](https://github.com/sharkdp/bat) configuration and themes.

| File | Description |
|------|-------------|
| `.config/bat/config` | bat config file |
| `.config/bat/themes/Catppuccin Frappe.tmTheme` | Catppuccin Frappe theme |
| `.config/bat/themes/Catppuccin Latte.tmTheme` | Catppuccin Latte theme |
| `.config/bat/themes/Catppuccin Macchiato.tmTheme` | Catppuccin Macchiato theme |
| `.config/bat/themes/Catppuccin Mocha.tmTheme` | Catppuccin Mocha theme |

`zsh/.zsh/plugins/cli.zsh` aliases `cat` to `bat` when available.
