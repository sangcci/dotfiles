# yazi

[Yazi](https://github.com/sxyazi/yazi) terminal file manager configuration.

Primary file manager. Launched via the `y` shell function from `zsh/.zsh/plugins/yazi.zsh`, which `cd`s into the directory yazi exits in.

| File | Description |
|------|-------------|
| `.config/yazi/yazi.toml` | General settings |
| `.config/yazi/keymap.toml` | Key bindings |
| `.config/yazi/theme.toml` | Active theme / flavor selection |
| `.config/yazi/flavors/catppuccin-mocha.yazi/` | Catppuccin Mocha flavor |
| `.config/yazi/flavors/darkvoid.yazi/` | Darkvoid flavor |
| `.config/yazi/flavors/kanagawabones.yazi/` | Kanagawabones flavor |
| `.config/yazi/flavors/kanso.yazi/` | Kanso flavor |
| `.config/yazi/flavors/vscode-light-modern.yazi/` | VS Code Light Modern flavor |

Current active flavor in `theme.toml`: `vscode-light-modern`.
