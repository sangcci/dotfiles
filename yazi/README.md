# yazi

[Yazi](https://github.com/sxyazi/yazi) terminal file manager configuration.

Primary file manager. Launched via the `y` shell function (defined in `zsh/functions.zsh`) which `cd`s into the directory yazi exits in.

| File | Description |
|------|-------------|
| `yazi.toml` | General settings |
| `keymap.toml` | Key bindings |
| `theme.toml` | UI theme |
| `flavors/kanso.yazi/` | Kanso Ink color flavor |
