# jankyborders

[JankyBorders](https://github.com/FelixKratz/JankyBorders) configuration for macOS window borders.

| File | Description |
|------|-------------|
| `.config/borders/bordersrc` | Border style, width, HiDPI, colors, and order |

Current defaults:

- Style: round
- Width: 2.0
- HiDPI: on
- Active color: bright white
- Inactive color: muted dark gray
- Order: below

AeroSpace also starts `borders` from `aerospace/.config/aerospace/aerospace.toml`.
